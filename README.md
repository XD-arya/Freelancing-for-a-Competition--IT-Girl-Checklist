# 💅 IT GIRL CHECK LIST

> *A luxury editorial e-commerce landing page for curated and affordable beauty & jewellery services.*

A dark, high-fashion single-page website featuring nail art, tattoos, jewellery, and curated kits — with a fully client-side cart powered by localStorage and UPI / WhatsApp integration.

---

## ✨ Features

- **Curated Kits** — Soft Girl, That Girl & Main Character bundles
- **Dynamic Cart** — quantity steppers, persistent via `localStorage`
- **WhatsApp Integration** — pre-filled design inquiry messages open directly in WhatsApp
- **UPI Checkout** — "Proceed to Clock It" triggers a Google Pay deep link with the exact cart total
- **Fully Responsive** — mobile-first design with a sticky bottom nav bar
- **No frameworks** — pure HTML + Tailwind CDN + vanilla JS
